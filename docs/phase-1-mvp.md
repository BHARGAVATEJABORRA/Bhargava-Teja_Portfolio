# Phase 1 MVP Notes

## Scope implemented
- Recruiter-first homepage with 10 IA sections.
- One hero wow interaction only (tilt-based signal card with reduced-motion support).
- Typed local content contracts for projects, experience, skills, and articles.
- Minimal Control Center focused on recruiting value.
- Contact flow with explicit API response contract.
- Baseline analytics hooks for core engagement events.
- Baseline SEO metadata, canonical URL, robots, sitemap, and JSON-LD.

## Deliberate deferrals (Phase 2/3)
- Expanded AI companion behavior and command logic.
- Additional wow interactions outside hero.
- Advanced Control Center modules.
- Personalization systems.

## Placeholder behavior
- Contact API logs submissions to server output only.
- External profile and case-study URLs are placeholders pending final profile links.
- Site URL in metadata is placeholder and should be replaced before production launch.
